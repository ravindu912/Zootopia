# Zootpia

Welcome to the Zootpia Android App! 🐾

## Features

- 🐶 Pet selection (Dog, Cat, Bird, Fish)
- 📅 Daily/Weekly Goals
- 🎓 Pet Training
- 🏆 Achievements
- 📱 Built with Android Studio
